# Linux Commands Demonstration

## Command 1: whoami
shows the current user

## COmmand 2: uptime
shows the time and days system has been running, user and load average of the current

## Command 3: uname -a
displays the unix distribution with some more information; date, time, user/group
 
## Command 4: id
dispalys the user id, the group id, the amount of users in a group and others

## Command 5: top
shows processes running in the kernel

## Command 6: dmesg | head -10
shows the first 10 lines when one runs the dmesg command (i dont really know what it does)

## Command 7: diff
shows the difference in the content between files

## Command 8: grep
searches for particular words or phrase in a file
